Projeto desenvolvido durante o curso de Desenvolvimento de Sistemas, com foco em lógica de programação, JavaScript, interface e organização de projeto.

Esta arena foi criada para praticar mecânicas de jogos 2D: movimentação, colisão, comandos especiais, HUD e telas de menu.

Versão atual com melhorias visuais, novos personagens e ajustes de usabilidade com base no feedback do jogador.
